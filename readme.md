# Keypress Counter
A simple keypress counter made in Python

## Getting Started
* Download the latest release `.exe `file or `git clone` this repo.
* Run the `.exe` file or `python main.py` after `pip install keyboard`
* Enter the key you want to track the number of key presses for, this creates a file to store the number of keypresses, which updates on every keypress.
* To close the program hit control c.
